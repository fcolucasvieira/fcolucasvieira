# 😎 Sobre mim

Sou desenvolvedor *Back-End* em formação, focado na construção de aplicações concretas utilizando *Java e Spring Boot*.

Atualmente estudo e desenvolvo projetos ligados a:

- APIs REST escaláveis
- Segurança com JWT e Spring Security
- Integrações entre sistemas com Spring Cloud
- Testes automatizados com JUnit e Mockito
- Containerização com Docker
- Comunicação assíncrona com RabbitMQ
- Arquitetura de Microsserviços
- Cloud e Infra (AWS)

💡 Tenho grande interesse em arquitetura de software, boas práticas, Clean Code e nas práticas que tornam aplicações mais confiáveis, escaláveis e sustentáveis ao longo do tempo.
